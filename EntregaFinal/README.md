|                                                  |
| Este proyecto consiste en obtener la ecuación    |
| polinomial característica de un sensor termopar. |
|                                                  |
|                                                  |
| Requisitos de instalación:                       |
| -Conexión a Internet                             |
| -El archivo instalador InstallerTermopares.exe   |
|                                                  |
|                                                  |
| Pasos:                                           |
| ***Abra el instalador InstallerTermopares.exe    |
|       Aparecerá el Wizard de Instalación de      |
|       MATLAB. Indicando el nombre de la          |
|       aplicación:                                |
|                                                  |
|          Caracterizacion_De_Termopares 1.0       |
|          Metodos Numericos                       |
|  *** De click en Next                            |
| 	Aparecerá la ventana : 'Choose installation| 
|       folder'                                    |
|  ***Elija la carpeta donde se instalará el       |
|     programa.                                    |
|  ***Elija si desea añadir un acceso directo en   |
|     escritorio.                                  |
|  Aparecerá la ventana de Matlab Runtime.         |
|  ***Elija la carpeta donde se instalará el módulo| 
|     MATLAB RUNTIME (Se recomienda dejar la       |
|     carpeta                                      |
|     por Default ).                               |
|  Si ya eligió la carpeta de instalación, o en su |
|  defecto, el módulo Runtime ya está instalado,   |
|  ***de click en Siguiente.                       |
|  ***En la ventana de Confirmation,               |
|     de click en Install.                         |
|     Espere a que se instale. Puede tardar unos   |
|     minutos.                                     | 
|  ***Abra la aplicación                           |
|          Caracterización_De_Termopares.exe       |
|  ***Disfrute                                     |
|                                                  |
